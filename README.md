# ColdWatch

MVP IoT para monitoreo de temperatura en neveras comerciales.

## Stack

- ESP32
- PlatformIO
- MQTT (futuro)
- PostgreSQL (futuro)

## Objetivo actual

Leer temperatura usando DHT22 y mostrarla por Serial cada 5 segundos.

## Requisitos técnicos

- Código modular
- Evitar delay()
- Fácil de escalar
- Buenas prácticas
